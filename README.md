# 🚀 QueueFlow AI

Smart Crowd Management System for large events like stadiums, concerts, and festivals.

---

## 📌 Problem
Managing long queues at large events is difficult. People waste time waiting without knowing which gate is faster.

---

## 💡 Solution
QueueFlow AI helps users:
- View wait times at different gates
- Get the fastest entry recommendation
- See live crowd alerts
- Interact with a smart assistant

---

## ⚡ Key Features

- 🔄 Dynamic Crowd Simulation (no backend required)
- 🚀 Smart Gate Recommendation (lowest wait time)
- 📊 Real-time Alerts System
- 💬 Simple Assistant for quick queries
- 🎯 Clean and Professional UI

---

## 🧠 How It Works

- Random data is generated for gate wait times
- System automatically:
  - Updates crowd status (Low / Medium / High)
  - Finds the fastest gate
  - Displays alerts
- Button click simulates real-time updates
